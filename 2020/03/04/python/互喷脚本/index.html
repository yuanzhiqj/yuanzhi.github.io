介绍

即一个能实现按下指定的任意键就能从文本库中调取一条语句并复制到剪贴板的功能。

希望大家还是能营造一个良好的网络环境，把这个代码当成一个学习过程就好啦。

遇到问题

如果遇到中文字符乱码的情况，也许是SetClipboardData()这里出现问题。我在测试的时候发现参数s的字符串类型为byte，所以decode解码成了’utf-8’的编码。如果在你的电脑上运行参数s的类型是str的话，就注释掉decode的那一条语句。
