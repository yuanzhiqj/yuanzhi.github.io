python/python学习笔记5函数

发表于 2020-02-08 | 分类于 python入门

定义函数

使用关键字def，定义以冒号结尾。

传递实参

位置实参

要求实参的顺序和形参的顺序相同

关键字实参

每个实参都由变量名和值组成，即传递给函数的是名称-值对

编写函数时，可以给每个形参指定默认值

def describe_pet(pet_name, animal_type='dog'):

注意：使用默认值时，在形参列表中必须先列出没有默认值的形参，再列出有默认值的实参。这让python能够正确地解读位置实参。

禁止函数修改列表

要将列表的副本传递给函数，可以这样做

function_name(list_name[:])

传递任意数量的实参

def function_name(*parameters)

符号*让python创建一个空元组，并将所有收到的值都封装到这个元组中。

发表于 2020-02-05 | 分类于 python入门

列表索引

将索引指定为-1，可让python返回最后一个列表元素。

同样，索引-2返回倒数第二个列表元素，索引-3返回倒数第三个列表元素，以此类推。

在列表中添加元素

1、在列表末尾添加元素

方法append(添加的元素)

2、在列表任意位置添加元素

方法insert(插入的位置，添加的元素)

从列表中删除元素

1、使用del语句删除元素

前提：知道要删除的元素在列表的位置

del list[index]

2、使用pop()方法删除元素

方法pop()可以删除列表末尾的元素，并且能够继续使用该元素

popped = list.pop()

增加参数可以弹出任意位置的元素

popped = list.pop(index)

3、根据值删除元素

方法remove(要删除元素的值)

该方法也可以继续使用它的值

*注意：如果列表中有重复的值，remove()只能删除第一个指定的值

组织列表

使用sort()对列表进行永久性排序

按照字母顺序排序。

如果需要按字母顺序相反的顺序排列元素，则需向sort()方法传递参数reverse=True。

list.sort(reverse=True)

使用sorted()对列表进行临时排序

print(sorted(list))

使用函数range()

函数range()让python从指定的第一个值开始计数，在达到第二个值后停止，因此输出不包括第二个值。

还可以指定步长even_numbers = list(range(2,11,2)) //打印1~10之内的偶数

使用list()创建数字列表

numbers = list(range(1,6))
